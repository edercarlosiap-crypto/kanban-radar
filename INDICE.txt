🚀 PROCESSO DE INÍCIO
═══════════════════════════════════════════════════════════════════════════

PASSO 1: LEIA A DOCUMENTAÇÃO
   1. Leia este arquivo (00_LEIA_PRIMEIRO.txt)
   2. Abra README.md para documentação completa
   3. Veja QUICKSTART.md para início rápido

PASSO 2: CONFIGURE .env
   Edite: backend/.env
   
   JWT_SECRET=sua_chave_secreta_super_segura_123456
   PORT=5000
   NODE_ENV=development

PASSO 3: INSTALE DEPENDÊNCIAS
   Terminal 1:
   cd backend
   npm install
   
   Terminal 2:
   cd frontend
   npm install

PASSO 4: INICIE SERVIDORES
   Terminal 1 (Backend):
   cd backend
   npm start
   → Porta 5000
   
   Terminal 2 (Frontend):
   cd frontend
   npm start
   → Porta 3000 (abre automaticamente)

PASSO 5: PRIMEIRO ACESSO
   1. Acesse http://localhost:3000/login
   2. Clique em "Registrar"
   3. Preencha seus dados
   4. Clique em "Registrar"
   5. Auto-login → Dashboard

BACKEND

📄 server.js
   • Inicia o servidor Express
   • Conecta ao banco de dados
   • Define porta
   • Exibe mensagem de sucesso

📄 app.js
   • Configura Express
   • Middleware: CORS, JSON, CORS, FileUpload
   • Registra rotas
   • Tratamento de erros

📄 config/database.js
   • Cria conexão SQLite
   • Cria tabelas automaticamente (usuarios e radar)
   • Funções auxiliares para queries (db_run, db_get, db_all)

📄 middleware/auth.js
   • Verifica token JWT
   • Extrai usuarioId do token
   • Protege rotas

📄 models/Usuario.js
   • Criar usuário com senha pré-criptografada
   • Buscar por email
   • Verificar senha
   • Gerar token JWT
   • Buscar por ID

📄 models/Radar.js
   • Criar item
   • Listar por usuário
   • Buscar específico
   • Atualizar campos
   • Deletar
   • Calcular status automático
   • Obter estatísticas

📄 controllers/authController.js
   • registrar() - POST /auth/register
   • login() - POST /auth/login
   • obterUsuario() - GET /auth/me

📄 controllers/radarController.js
   • listar() - GET /radar
   • criar() - POST /radar
   • buscar() - GET /radar/:id
   • atualizar() - PUT /radar/:id
   • deletar() - DELETE /radar/:id
   • importarExcel() - POST /radar/importar-excel
   • obterEstatisticas() - GET /radar/estatisticas/dashboard

📄 routes/authRoutes.js
   • POST /auth/register

🔑 CHAVES IMPORTANTES
═══════════════════════════════════════════════════════════════════════════

Banco de Dados:
   • Arquivo: backend/database.db
   • Tipo: SQLite
   • Tabelas: usuarios, radar
   • Criado automaticamente ao iniciar

Autenticação:
   • Método: JWT
   • Expiração: 7 dias
   • Criptografia Senha: bcrypt (10 rounds)
   • Token armazenado em: localStorage.getItem('token')

API:
   • Base: http://localhost:5000
   • Formato: JSON
   • Autenticação: Bearer TOKEN

Frontend:
   • Framework: React 18
   • Roteamento: React Router v6
   • HTTP: Axios
   • Drag Drop: @hello-pangea/dnd
   • Styling: CSS customizado
