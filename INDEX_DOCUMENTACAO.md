## 💡 CASOS DE USO

### Caso 1: "Preciso criar um sistema de gestão de leads"
```
1. Abra: PROMPT_RAPIDO.md
2. Copie o prompt
3. Substitua: "um sistema de gestão de leads com 
   campos: nome, email, telefone, empresa, status"
4. Use com IA para gerar estrutura
5. Implemente detalhes com TEMPLATES_CODIGO.md
```

### Caso 2: "Estou migrando projeto legado"
```
1. Abra: CHECKLIST_COMPATIBILIDADE.md
2. Marque o que já tem no projeto atual
3. Identifique gaps
4. Use PROMPT_ARQUITETURA.md para entender o que falta
5. Implemente faltantes com TEMPLATES_CODIGO.md
```

### Caso 3: "Preciso adicionar autenticação"
```
1. Abra: TEMPLATES_CODIGO.md
2. Copie: middleware/auth.js
3. Copie: PrivateRoute.js
4. Copie: Auth Helpers
5. Configure JWT_SECRET no .env
```

### Caso 4: "Preciso adicionar Excel import/export"
```
1. Instale: xlsx ^0.18.5
2. Abra: TEMPLATES_CODIGO.md
3. Copie: Export Excel Template (frontend)
4. Copie: Import Excel Template (backend)
5. Adapte campos conforme necessário
```
