━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

AUTENTICAÇÃO (Públicas)
   POST   /auth/register          → Registrar usuário
   POST   /auth/login             → Login

AUTENTICAÇÃO (Protegidas)
   GET    /auth/me                → Dados do usuário logado

RADAR (Todas protegidas por JWT)
   GET    /radar                  → Listar itens
   POST   /radar                  → Criar item
   GET    /radar/:id              → Buscar item
   PUT    /radar/:id              → Atualizar item
   DELETE /radar/:id              → Deletar item
   POST   /radar/importar-excel   → Importar Excel
   GET    /radar/estatisticas/... → Estatísticas

"Porta 5000 já em uso"
   → Feche outro programa na porta 5000
   → Ou mude PORT em .env

"CORS error"
   → Backend deve estar em http://localhost:5000
   → Frontend deve estar em http://localhost:3000

"Banco não inicializa"
   → Delete backend/database.db
   → Reinicie o backend

"Cannot GET /login"
   → Verifique se o backend está rodando
   → Acesse http://localhost:5000/health para testar
